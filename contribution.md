Voir la section [open issues][issues-url] pour accéder à la liste des capacités nouvelles identifiées.

Les contributions sont ce qui fait que la communauté open source est un endroit à ce point incroyable
pour apprendre, inspirer et créer. Toute contribution que vous pouvez faire est **fortement appréciée**.
  
Si vous avez des suggestions pour améliorer cette ressource, merci de clôner \[eng. *Fork*\] 
le référentiel Github et créer une demande de fusion de code \[eng. *pull request*\]. Vous pouvez aussi tout
simplement ouvrir un ticket \[eng. *issue*\] avec le tag **enhancement**.

N'oubliez pas d'ajouter une étoile au projet ! Merci d'avance.
